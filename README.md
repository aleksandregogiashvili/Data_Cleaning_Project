# Data_Cleaning_Project

MySQL Data Cleaning Project

Overview

This project demonstrates the process of cleaning a real-world layoffs dataset using MySQL. The objective was to prepare messy data for further analysis by identifying duplicates, handling missing values, standardizing inconsistent entries, and improving overall data quality.

Skills Demonstrated:
-- MySQL
-- Data Cleaning
-- Data Transformation
-- UPDATE statements
-- ALTER TABLE
-- DELETE statements
-- String Functions
-- Date Formatting
-- Cleaning Steps
-- Removed duplicate records using ROW_NUMBER().
-- Standardized inconsistent values
-- Converted date values into a proper DATE format.
-- Replaced blank values with NULL where appropriate.
-- Filled missing values using related records.
-- Removed unnecessary columns after cleaning.

The dataset was transformed into a clean, standardized version suitable for exploratory data analysis and reporting.
